# Coupons (`@loyaltyos/coupons`)

Coupon management — create, distribute, and track discount codes.

## Discount Types

- **Percentage** — % off on purchase
- **Fixed** — fixed amount off
- **Free Product** — free item with purchase
- **Free Shipping** — free shipping on order
- **Extra Points** — bonus points on purchase
- **Experience** — access to exclusive events or content

## Modes

- **Shared** — one code used by many members
- **Individual** — unique code per member
- **Limited** — first N uses across all members

## Features

- Bulk code generation
- Usage tracking with per-member redemption logs
- Configurable expiry
- Stacking control per coupon

See the [full README on GitHub](https://github.com/jvillatox/loyaltyos/blob/main/packages/coupons/README.md) for details.
