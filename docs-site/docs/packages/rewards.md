# Rewards (`@loyaltyos/rewards`)

Rewards catalog — items members exchange points for.

## Categories

- **Discount** — percentage or fixed discount coupons
- **Physical Product** — shipped merchandise
- **Gift Card** — digital gift cards
- **Experience** — exclusive events or access
- **Charity** — donate points to a cause
- **Coalition Transfer** — convert points to coalition program

## Features

- Points cost with optional tier gating
- Stock management with restock API
- Eligibility checks (points balance, tier requirement, stock availability)
- Idempotent redemption
- Redemption history and analytics

See the [full README on GitHub](https://github.com/jvillatox/loyaltyos/blob/main/packages/rewards/README.md) for details.
